## ✨ Features

- 🌈 Only calculates your IMC based on:
  - 📦 Your Weight
  - 🛡 Your Height
- 🌍 Then Gives you a insight of how your IMC are, and if you need to see a doctor.

## 🖥 Environment Support

- Mobile Devices

## 📦 Install

```bash
git clone https://github.com/Dev-Shinsei/imcMobile
cd imcMobile
yarn
yarn start
```
